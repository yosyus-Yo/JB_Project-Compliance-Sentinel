# 전문가 심의 지식 업로드 예시 — 자동차 할부/대환대출 광고

## 1. 심의관 절차 체크리스트

심의관 체크리스트: 먼저 상품유형(대출/할부/예금/투자), 채널(배너/앱푸시/SNS/랜딩), 대상 고객을 분류한다. 다음으로 금지 표현과 필수 고지 누락을 분리해 판단한다. 수정안은 원문보다 보수적으로 작성하고, 심사 조건·한도·금리 범위·상환 조건을 빠뜨리지 않는다.

## 2. 내부 기준 원문 / RAG 근거

내부 기준 원문 2026: 자동차 할부 및 대환대출 광고에서 "당일 무조건 승인", "신용점수 무관", "한도 무제한", "무심사" 표현은 소비자가 심사 절차와 거절 가능성을 오인할 수 있으므로 고위험 또는 반려 후보로 분류한다. 대출성 상품 광고는 심사 조건, 금리 범위, 상환 조건, 신용도 영향, 중도상환수수료 여부를 필수 고지해야 한다.

## 3. 반복 경험 사례 / 장기 메모리 후보

반복 경험 사례: JB우리캐피탈 자동차 할부 랜딩페이지에서 '당일 무조건 승인' 문구가 등장하면 과거 반려 사례와 동일하게 critical 후보로 본다. 이 패턴은 승인 전까지 pending memory로 두고, 준법 담당자 승인 후 readonly 장기 메모리로 승격한다. 문의 전화 010-1234-5678은 저장 전에 반드시 마스킹되어야 한다.
