## 주의 (제출 전 확인)

- ⚠️ 공식 요강의 **파일 형식 / 해상도 / 러닝타임 하한 / 자막 의무**를 본 스크립트 적용 전 재확인.
- ⚠️ 내레이션 중 "법령정보센터", KB 건수, "다크패턴 2026.4" 등 수치·출처는 제출 시점 기준 실제와 일치하는지 확인(허위·과장 표현은 감점·신뢰 훼손).
- ⚠️ **"Qdrant 하이브리드 검색"·"3대 도구 통합 확보"는 과장 위험** — 현재 RAG는 키워드 fallback 모드, AgentLoop/AgentCompiler 본체는 외부 도구다. 심사 질의 시 "**핵심 거버넌스 기능(보안·예산·비용)을 런타임 자체 구현, 외부 전문 도구는 선택 연계, 도구 없어도 fallback 동작**"으로 답한다. AgentLoop의 drift 감시는 실시간이 아니라 배포 게이트에 배치했다고 설명.
- ⚠️ **자가교정 루프백(최대 3회)은 LLM 런타임 ON 시에만 실질 동작** — `CS_ENABLE_LLM_RUNTIME=1` 미설정(deterministic) 시 수정안 미생성 채로 retry만 증가하고 HITL로 전환됨. CUT 3-4 펀드 시연 전 `revise_trace`에 `attempt 1·2·3`이 찍히는지 리허설 확인.
- ⚠️ 6인 보드 페르소나명/근거는 `src/compliance_sentinel/board.py` 기준 — 변경 시 자막도 동기화.
- ⚠️ 위험등급/하이라이트/위반유형 라벨/수정원고는 **실제 심의 결과 화면을 녹화**해 그대로 사용(스크립트의 등급·표현은 예시이므로 실제 출력과 다르면 실제 출력을 따른다).
